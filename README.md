# 📊 Sales & Product Performance Dashboard (Power BI)

## 🌟 Overview  
This project showcases an interactive **Power BI dashboard** built to analyze sales performance, revenue trends, and product insights. It helps in making data-driven decisions using visual analytics.

---

## 🚀 Key Metrics  
- 💰 Total Revenue: 198M+  
- 📈 Total Profit: 85M+  
- 📦 Sold Quantity: 8.9M+  
- 🧾 Total Transactions: 228K+  
- 👥 Total Customers: 402  

---

## 📊 Features  
- 📈 Revenue analysis by fiscal year (2013–2016)  
- 🧑‍💼 Employee-wise performance tracking  
- 🌍 Geographic analysis (state & city level)  
- 📦 Product insights (size & color)  
- 🔍 Transaction-level detailed data  
- 🎛️ Interactive filters (Year, Color, Buying Group)  
- 🔄 Toggle between Summary & Product Details views  

---

## 💡 Key Insights  
- 2015 is the best-performing year (~62M revenue)  
- Top employees contribute significantly to revenue  
- Certain product categories dominate sales  
- Regional performance varies across locations  

---

## 🛠️ Tech Stack  
- Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Data Modeling & Visualization
